# SKILL-TRACKER

Full-stack Skill Tracking Web Application built with the MEAN stack (MongoDB, Express, Angular 17, Node.js).

## Features
- 🛡️ **Admin Dashboard**: Platform-wide metrics and user moderation.
- 🚀 **Skill Management**: Track any domain (Coding, Music, Yoga, etc.) with custom categories.
- 🎯 **Goals & Roadmap**: Set future learning targets with date validation.
- 🏆 **Achievements**: Automatic badge system based on progress streak.
- 📊 **Weekly Activity**: Visual chart tracking of learning time.
- 🌓 **Dark Mode**: Premium UI with glassmorphism and theme switching.

## Deployment
This app is ready for cloud deployment.
1. Build frontend: `cd frontend && npm run build`
2. Configure `.env` in backend.
3. Start backend: `cd backend && npm start`
